### Docker commands:
* For run:
$ docker run ifo/dva
* For check, run or not:
$ docker ps
* For stop:
$ docker stop <conatiner-id>
* For docker image delete:
$ docker rmi <conatiner-id>
(-f for force to delete)

### Docker-compose commands:
* For run:
$ docker-compose up
* For check, run or not:
$ docker-compose ps
* For stop:
$ docker-compose down
